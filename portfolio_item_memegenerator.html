Using the Top 100 Memes Meme Generator

Created: January 13, 2021

The meme generator is a fun and whimsical project idea of mine that a popular trend, and makes it easier and more accessible for people to use. The meme generator uses an API from imgflip that returns 100 of the current most popular meme templates.

More information about the API can be found here: https://imgflip.com/api

This project was a great learning opportunity as it allowed me to practice useState, and useEffect which are the most fascinating parts of REACT.

By pressing the get a new meme image, this application will randomly select from the current 100 best meme images from the API. Then update the image on the screen to the corresponding image.

The API returned the following image from Pirates of the Caribbean, as a meme template. This was performed around the time of the Johnny Depp v Amber Heard Trial, thus it stands to reason that this image was a by-product of current relavence in the world.

Obviously just simply returning an image would not be of much use. Thus the user can type in any text in the two fields above, and the will update live, on the image. An example is shown below:

Through the project I learned, how to update components live, and avoid unnecessarily re-rendering a page when no changes have been made. React useState, and useEffect were instrumental in this project, and I will be incorporating them in all my future applications that can benefit from their use.
