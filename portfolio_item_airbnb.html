REACT Clone AirBnb

Created: June 6, 2021

The AirBnb project was a great opportunity to create a dynamic clone that matches a known website.

It served as the foundation blocks for my interest in developing in REACT. It was a great challenge to try and re-create a website from scratch, not even initally having various assets used by the developers. One of the hardest challenges was to pin-point the small hard to notice details. This could be spacing between various components, or even something simple as the weight of a font.

The website is a static page, but it adapts with new data presented. As new data is inserted into the data file, REACT will automatically map the data to an object. Then create a card component using the data.
